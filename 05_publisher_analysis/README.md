## Коэффициенты для изменения ставок (НОВОЕ):
- `publishers_bid_coefficients.csv` - коэффициенты для каждого паблишера с учетом целевой CPA по форматам

### Методология расчета:
1. Находим среднюю CPA для PLR/NOR кампаний внутри каждого формата
2. Вычитаем 30% для получения целевой CPA
3. Рассчитываем коэффициент: **Целевая CPA / Текущая CPA**
4. Умножаем текущую ставку на коэффициент = новая ставка

### Целевые CPA по форматам (-30% от среднего):
- **POP**: $0.096 (текущий avg: $0.140)
- **PUSH**: $0.014 (текущий avg: $0.023)
- **VIDEO**: $0.047 (текущий avg: $0.075)
- **BANNER**: $0.005 (текущий avg: $0.007)
- **NATIVE**: $0.024 (текущий avg: $0.137)

### Примеры коэффициентов:
- **Коэф > 1.3**: УВЕЛИЧИТЬ ставку на 30%+
- **Коэф 1.1-1.3**: Увеличить на 10-30%
- **Коэф 0.9-1.1**: Оставить без изменений
- **Коэф 0.7-0.9**: Снизить на 10-30%
- **Коэф < 0.7**: СНИЗИТЬ ставку на 30%+
