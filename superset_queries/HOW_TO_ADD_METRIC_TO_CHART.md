# Как добавить метрику (количество реактиваций) в Chart

Если Chart показывает только периоды неактивности, но не показывает количество, нужно добавить метрику.

## Решение:

### Шаг 1: Откройте Chart для редактирования

1. На Dashboard найдите Chart "Reactivations all"
2. Нажмите на иконку **"..."** (три точки) в правом верхнем углу Chart
3. Выберите **"Edit chart"** или **"Edit"**
4. Или нажмите на название Chart

### Шаг 2: Добавьте метрику COUNT(user_id)

1. В Chart Builder в правой панели найдите раздел **"Metrics"**
2. Нажмите **"+ Add"** или **"Drop columns/metrics here or click"**
3. Выберите колонку **`user_id`** из списка
4. Выберите функцию: **"COUNT"** или **"COUNT DISTINCT"**
   - **COUNT DISTINCT** - подсчитывает уникальных пользователей (рекомендуется)
   - **COUNT** - подсчитывает все записи
5. Метрика должна появиться в списке как `COUNT(user_id)` или `COUNT DISTINCT(user_id)`

### Шаг 3: Проверьте Group by

Убедитесь, что в разделе **"Dimensions"** (или **"Group by"**) выбрано:
- **`inactivity_period`**

### Шаг 4: Обновите Chart

1. Нажмите **"Update chart"** или **"Run"**
2. Chart должен показать:
   - **7-14 days**: X реактиваций
   - **14-30 days**: Y реактиваций
   - **30-90 days**: Z реактиваций
   - **90+ days**: W реактиваций

### Шаг 5: Сохраните Chart

1. Нажмите **"SAVE"** (вверху справа)
2. Chart обновится на Dashboard

## Альтернативный способ (если метрика не добавляется):

1. В разделе **"Metrics"** нажмите **"+ Add metric"**
2. Выберите **"Custom SQL"** или **"SQL"**
3. Введите: `COUNT(DISTINCT user_id)`
4. Или введите: `COUNT(user_id)`
5. Нажмите **"Save"** для метрики
6. Нажмите **"Update chart"**

## Что должно быть в настройках Chart:

- **Dimensions (Group by)**: `inactivity_period`
- **Metrics**: `COUNT(user_id)` или `COUNT DISTINCT(user_id)`
- **Filters**: `first_deposit (No filter)` - оставьте как есть

После этого Chart будет показывать количество реактиваций для каждого периода неактивности.
